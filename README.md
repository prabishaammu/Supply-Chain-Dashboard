Supply Chain Performance Dashboard - Power BI

Objective
Analyze supply chain performance across 180,000+ orders including 
sales trends, delivery performance and regional analysis.

Dataset
Source: DataCo Supply Chain Dataset
Download from: https://www.kaggle.com/datasets/dataseeker/supply-chain-dataset
- Records: 180,519 rows
- Columns: 53 columns

Tools Used
- Power BI Desktop
- Power Query
- DAX
  
Dashboard Pages
1. Overview - KPI cards, Orders by Region, Orders by Status
2. Sales Analysis - Sales trend by year, Top products by sales
3. Delivery Performance - Late delivery rate, Delivery status breakdown

Key Insights
- Total Sales: $36.78M across all regions
- 54.83% Late Delivery Rate — major area for improvement
- Sales peaked in 2016 and declined through 2018
- Central America and Western Europe are top performing regions
- Average delivery time is 3 days
